# raw/

> **LLM 只读目录。本目录下任何文件禁止修改。**

本目录存放本项目的**原始素材**,包括但不限于:

- 方案设计文档(如 `plan-1.md` —— 双模型编排方案的原始设计)
- 对话记录(如日期命名的会话记录,记录当次落地/验证过程)
- 后续可能追加的:网关/vLLM 配置片段、测试日志、排错记录

## 命名约定

- 方案设计文档:`<方案名>.md`(如 `plan-1.md`)
- 对话记录:`<YYYY-MM-DD>-conversation.md`(如 `2026-07-04-conversation.md`)
- 排错/日志类:`<主题>-<MMDD>.md`

## LLM 行为约束

1. **只读** — 用 Read / grep / cat,不要 Edit / Write 任何 raw 下文件
2. **发现问题只注 wiki** — 发现 raw 错别字、矛盾、格式问题 → 在对应 wiki 主题页加 `> 已过时:YYYY-MM-DD 起 raw/X 处有 Y 问题`
3. **新文件入仓必走 Ingest 工作流** — 详见 [[../wiki/工作流-ingest-query-lint#一-ingest-工作流]]
4. **保留原始组织结构** — 不要散开重命名已有文件
